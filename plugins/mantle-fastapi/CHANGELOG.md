# Changelog

All notable changes to the mantle-fastapi plugin will be documented in this file.

## [1.0.0] - 2026-02-08

### Added

- **fastapi-reviewer** agent: Quality standards reviewer for FastAPI applications covering Pydantic models, route organization, dependency injection, response models, error handling, OpenAPI documentation, security patterns, and database session management. PASS/FAIL output.
- **fastapi-async** agent: Concurrency reviewer for async/await patterns, blocking operations, BackgroundTasks, dependency lifecycle, database async patterns, connection pool management, and startup/shutdown events.
- **python-lint** agent: Linter agent using ruff check, ruff format, and mypy for error, formatting, and type checking reports.
- **fastapi-best-practices** skill: Comprehensive FastAPI best practices guidance with references on Pydantic, dependency injection, async patterns, Alembic migrations, and testing.
