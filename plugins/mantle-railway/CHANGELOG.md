# Changelog

All notable changes to the mantle-railway plugin will be documented in this file.

## [1.0.0] - 2026-02-08

### Added

- **railway-reviewer** agent for auditing Railway deployment configurations
  - Reviews Nixpacks configuration, environment variables, health checks, scaling, volumes, networking, and more
  - Produces PASS/FAIL output with detailed findings and remediation steps
- **railway-best-practices** skill with reference documentation
  - Nixpacks configuration and build customization
  - Environment and variable management
  - Service networking and domain configuration
  - Deployment strategies, scaling, and operations
